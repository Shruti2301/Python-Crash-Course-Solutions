## ▶️ Running the Code

Clone the repository:

```bash
git clone https://github.com/Shruti2301/Python-Crash-Course-Solutions.git
```

Navigate to the project:

```bash
cd Python-Crash-Course-Solutions
```

Run any exercise:

```bash
python filename.py
```
